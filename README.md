# FireWatcher

Sistema preditivo de monitoramento de queimadas desenvolvido como projeto acadêmico. O objetivo do FireWatcher é utilizar dados ambientais e sensoriamento remoto para identificar regiões com alto risco de incêndio antes que os focos aconteçam, permitindo ações preventivas mais eficientes.

## Sobre o Projeto

Atualmente, grande parte dos sistemas de monitoramento de queimadas atua de forma reativa, identificando incêndios apenas após seu início. O FireWatcher propõe uma abordagem preditiva, utilizando variáveis ambientais e dados de satélite para estimar a probabilidade de ocorrência de queimadas com antecedência.

O projeto foi desenvolvido como atividade acadêmica com foco em inovação, sustentabilidade e aplicação de tecnologia para apoio à preservação ambiental.

## Objetivos

* Antecipar riscos de queimadas utilizando dados ambientais.
* Auxiliar órgãos ambientais na tomada de decisão.
* Promover ações preventivas antes do surgimento dos focos de incêndio.
* Demonstrar a aplicação de Inteligência Artificial em problemas ambientais.
* Conscientizar sobre os impactos das queimadas nos biomas brasileiros.

## Conceitos Utilizados

O FireWatcher foi idealizado considerando variáveis frequentemente utilizadas em estudos de monitoramento ambiental:

* Temperatura da superfície terrestre;
* Índices de vegetação (NDVI);
* Umidade da vegetação;
* Histórico de precipitação;
* Condições climáticas;
* Sensoriamento remoto por satélites.

Essas informações podem ser utilizadas por modelos preditivos para identificar regiões com maior probabilidade de ocorrência de incêndios florestais.


Ou utilize uma extensão como Live Server no VS Code para executar localmente.
